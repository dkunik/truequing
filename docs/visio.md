# Visión

## ¿Qué es Truequing?

Truequing es una herramienta abierta para facilitar el intercambio de figuritas.

Su objetivo es ayudar a las personas a encontrar oportunidades de intercambio antes y durante los encuentros de coleccionistas, utilizando información colectiva construida por la propia comunidad.

## El problema

Cuando un álbum comienza a completarse, la cantidad de figuritas repetidas suele crecer mucho más rápido que la cantidad de figuritas faltantes.

En ese momento el desafío deja de ser conseguir figuritas y pasa a ser encontrar personas con quienes intercambiar.

En encuentros masivos, plazas, parques o reuniones de coleccionistas, una gran parte del tiempo se invierte en buscar posibles intercambios.

Truequing busca reducir ese tiempo y facilitar el contacto entre personas con intereses compatibles.

## Nuestra propuesta

Truequing ayuda a:

- Encontrar quién tiene las figuritas que buscás.
- Identificar posibles intercambios.
- Mantener actualizado el estado de un álbum.
- Facilitar intercambios presenciales mediante herramientas offline.
- Construir estadísticas colectivas sobre disponibilidad y demanda.

## Información colectiva

Cada usuario que participa aporta información que ayuda a toda la comunidad.

La suma de faltantes y repetidas permite construir conocimiento colectivo sobre:

- Disponibilidad de figuritas.
- Demanda de figuritas.
- Dificultad relativa para conseguir determinadas figuritas.
- Oportunidades de intercambio.

El objetivo es poner esa información a disposición de toda la comunidad.

## Principios

### Comunidad

Truequing busca fortalecer las comunidades de intercambio existentes.

### Software libre

El proyecto se desarrolla como software libre y abierto.

### Acceso libre

Las funcionalidades básicas deben estar disponibles para cualquier persona interesada en intercambiar figuritas.

### Transparencia

La plataforma busca mostrar información útil para que las personas tomen decisiones, no reemplazar el criterio de quienes intercambian.

### Simplicidad

La herramienta debe reducir el esfuerzo necesario para encontrar oportunidades de intercambio.

## Alcance

Truequing es una herramienta para facilitar encuentros, contactos e intercambios entre coleccionistas.

Su propósito es ayudar a las personas a encontrarse, compartir información y construir comunidad alrededor del intercambio de figuritas.

## Origen

Truequing nació en La Plata como un proyecto comunitario desarrollado para ayudar a coleccionistas a encontrar oportunidades de intercambio de manera más eficiente.
